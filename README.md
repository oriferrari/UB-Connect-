# UB Connect+

Sistema universitario responsive para la gestión académica de alumnos, materias, comisiones, profesores, cursadas, notas y asistencia.

## Tecnologías utilizadas

- PostgreSQL / Supabase
- SQL
- Visual Studio Code
- React + Vite
